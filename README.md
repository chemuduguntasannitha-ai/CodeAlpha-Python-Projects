# CodeAlpha Python Internship Projects

This repository contains the projects completed as part of the CodeAlpha Python Programming Internship.

## Task 1: Exploratory Data Analysis (EDA)

* Performed data cleaning and analysis on the Titanic dataset.
* Calculated survival rates and explored relationships between features.
* Generated visualizations and statistical insights.

## Task 2: Interactive Dashboard

* Developed an interactive dashboard using Dash and Plotly.
* Created dynamic charts and data visualizations.
* Enabled user interaction for data exploration.

## Task 3: Sentiment Analysis

* Performed sentiment analysis on customer reviews using TextBlob.
* Classified reviews as Positive or Negative.
* Calculated sentiment polarity scores and generated charts.

## Task 4: Web Scraping

* Built a web scraper using BeautifulSoup and Requests.
* Extracted quotes, authors, and tags from a website.
* Saved the scraped data to CSV files for analysis.
